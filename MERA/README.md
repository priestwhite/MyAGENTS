# MERA: MCP-Enhanced Reasoning Algorithm

> AI 記憶系統設計專案

---

## 專案簡介

MERA (MCP-Enhanced Reasoning Algorithm) 是一個針對 **LocalMemoMCP** 長期記憶工具的設計討論與規範制定專案。

目標：建立一套可落地的「記憶治理規範」，讓 AI Agent 能更有效地儲存、檢索與管理長期記憶。
